1. Clona este repositorio o descarga el archivo `ej1.java`
2. Asegúrate de tener instalado el JDK (Java Development Kit)
3. Compila el archivo usando:
   ```bash
   javac ej1.java
   ```
4. Ejecuta el programa generado:
   ```bash
   java ej1
   ```
